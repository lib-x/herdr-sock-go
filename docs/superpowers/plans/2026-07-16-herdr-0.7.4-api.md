# Herdr 0.7.4 Go Socket API Adaptation Implementation Plan

> **For agentic workers:** Execute inline in the current repository. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add backward-compatible typed Go support for Herdr 0.7.4's JSON SockAPI and publish module tag `v0.7.4`.

**Architecture:** Extend existing constants, params, response types, event helpers, and `Client` methods without removing 0.7.3 fields. Keep JSON graphics and popup/plugin pane support in the existing newline-delimited request path; exclude the separate binary graphics stream.

**Tech Stack:** Go 1.22, standard library JSON/net/context packages, Unix socket fixtures, go-winio on Windows.

## Global Constraints

- Keep `CurrentProtocol` at 16 and add `CurrentHerdrVersion = "0.7.4"`.
- Preserve every existing exported identifier.
- Retain removed `CustomStatus` fields with `Deprecated:` comments.
- Use `map[string]*string` for metadata token patches.
- Do not implement `pane.graphics.stream`.
- Run `go test -race ./...` before publishing.
